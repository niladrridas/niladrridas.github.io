Raspberry Pi 4 IoT with Screen

This project was undertaken during my third year in the Diploma program in Electronics and Communication Engineering at Lovely Professional University (LPU). I utilized a Raspberry Pi 4 Model B and a microSD card to store the latest Raspbian OS, interfacing a 3.5-inch LCD touchscreen to operate the functions. During the installation process, I employed a pre-programmed LCD Python script to enable operation via the Bash shell in the terminal. This project earned me recognition at LPU with an outstanding grade and led to a publication on arXiv. Additionally, this project enhanced my foundational understanding of Linux and its basic shell commands. Subsequently, I also integrated Ubuntu OS into the hardware setup.

Voice Generation Chatbot

After transitioning to Computer Science through lateral entry at Lovely Professional University (LPU), I embarked on numerous micro projects. By researching solutions online and consulting various books, I utilized Python, Visual Studio Code, and libraries such as face_recognition, TextBlob, gTTS, and playsound to integrate functionality into the algorithms. Gradually, I enhanced the Python script to train the model for various tasks, including capturing audio from the microphone and converting it to text using Google's Web Speech API, analyzing the sentiment of user input using TextBlob, converting response text to speech using gTTS, and generating diverse responses based on user input, sentiment, and knowledge.
